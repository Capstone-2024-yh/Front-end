# Front-end
## 개요
이 프로젝트는 React.js를 이용해 생성된 프로젝트다.

## 실행 방법
1. Git bash에서 아래와 같이 입력하여 리포지토리를 클론한다.
   
   ```sh
   git clone git@github.com:Capstone-2024-yh/Front-end.git
   ```

2. VSCode, IntelliJ 등 IDE에서 해당 프로젝트와 터미널을 연다.
3. 터미널에서 아래와 같이 입력한다.

   ```sh
   cd ./frontend/              // 위치 이동
   npm install package.json    // 의존성 파일을 다운로드
   ```

4. 프로젝트를 실행한다.

   ```sh
   npm start
   ```
